== Description ==

This Alexa plugin allows you to share your current alexa rank!

The setup for this plugin is very easy and quick - Simply enable the plugin in your Wordpress admin panel and then enable the widget. Your alexa rank will then be shown on your blog for your visitors.

== Installation ==

1. Upload `/jr-alexa.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the JR Alexa Settings in your Administration Panel
4. Done!
